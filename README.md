# TapTone

TapTone is a Flutter app that helps colorblind users distinguish colors better.

## Features

- Apply assistive color filters to photos for deuteranomaly, protanopia, and tritanopia.
- Take a photo or select an image from the gallery.
- Detect a color name and hex code using the live camera.
- Preview color vision simulations using the live camera.

## Disclaimer

TapTone is an assistive tool, not a medical device. It helps enhance color differences, but results may vary depending on lighting conditions, camera quality, screen calibration, and individual color vision. TapTone does not diagnose, treat, cure, or prevent any medical condition. Do not rely on TapTone for medical, safety-critical, or emergency decisions.

## Development

TapTone is built with Flutter and Dart for iOS and Android.
